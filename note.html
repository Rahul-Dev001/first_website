HTML Tutorial

What is HTML ?

HTML, or HyperText Markup Language, is the standard markup language used to create and structure web pages. It defines the content and layout of web pages using a system of tags and elements that web browsers interpret to display information, including text, images, links, and multimedia. Essentially, HTML provides the foundational structure for nearly all content presented on the World Wide Web.

What is CSS ?

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in a markup language, most commonly HTML. It is a cornerstone technology of the World Wide Web, working alongside HTML and JavaScript to create interactive and visually appealing web pages.

    Key aspects of CSS:
  • Separation of Concerns: CSS promotes the separation of content (HTML) from presentation (CSS). This makes websites easier to maintain, update, and adapt to different devices and screen
  • Styling Capabilities: CSS allows developers to control a wide range of visual properties, including colors, fonts, spacing, layout, positioning, backgrounds, and even animations.
  • Cascading and Specificity: The "Cascading" in CSS refers to how styles are applied when multiple rules target the same element. A set of rules, including importance, specificity, and source order, determines which styles take precedence.
  • Selectors: CSS uses selectors to target specific HTML elements for styling. These can include tag names, class names (prefixed with a dot), IDs (prefixed with a hash), attributes, and more complex combinations.
  • Efficiency: By defining styles in external stylesheets, CSS allows for consistent styling across multiple web pages, reducing code duplication and making global changes more efficient.

HTML Boilerplate Code

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <title>My Web Page</title>

    <!-- Optional: Link your CSS file -->
    <link rel="stylesheet" href="style.css">

    <!-- Optional: Favicon -->
    <link rel="icon" href="favicon.ico" type="image/x-icon">
  </head>

  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>

    <!-- Optional: Link your JavaScript file -->
    <script src="script.js"></script>
  </body>
</html>

TAGS

If you want to build a beautiful website, tags are essential elements that help you achieve that. An HTML tag acts as a container for content or other HTML tags. Tags are words enclosed within <and> angle brackets. They serve as keywords that instruct the web browser on how to format and display the content.

DOCUMENT TAGS

  • <DOCTYPE html>: Specifies the document type.
  • <HTML>: Encloses the entire HTML Document.
  • <Head>: Contains meta-information and links to scripts and stylesheets.
  • <Body>: Contains the content of the web page.
